# Anime Streaming Website using ReactJS, Typescript and Firebase

## Basic setup

### Clone this project

```
git clone https://github.com/tlos3r/Anifufu.git
```

###

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
